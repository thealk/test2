3 RMarkdown crash course

But first: A crash course in RMarkdown

In Spring 2018 at R-Ladies #LdnOnt we practiced making a manuscript using RMarkdown (slides). Today we will up the ante and write a whole1 dissertation using RMarkdown!

bookdown is a brilliant R package designed to incorporate multiple RMarkdown files into a single final product, a book. But bookdown is not just for books:

Despite the package name containing the word “book”, bookdown is not only for books. The “book” can be anything that consists of multiple R Markdown documents meant to be read in a linear sequence, such as course handouts, study notes, a software manual, a thesis, or even a diary.
